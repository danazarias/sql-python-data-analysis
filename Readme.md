📊 Projeto Meteora: Inteligência de Dados para Decisões Estratégicas
Este projeto aplica técnicas de Data Science e SQL para analisar o desempenho da Meteora, uma loja de varejo com presença em todo o Brasil. O foco é transformar bases de dados em informações acionáveis para guiar as promoções, bônus salariais e estratégias de expansão da marca.

🎯 Contexto de Negócio (Cenário Fictício)
Para o planejamento do ano de 2021, a diretoria da Meteora definiu dois pilares prioritários:

Reconhecimento do Time: Analisar o desempenho dos vendedores em 2019 e 2020 para aplicar bônus e promoções meritocráticas.

Expansão em São Paulo (SP): Validar se SP é realmente a região mais propícia para novos investimentos e selecionar os dois melhores vendedores da região para liderar essa frente.

🛠️ Tecnologias Utilizadas
SGBD: SQLite para armazenamento e estruturação das tabelas.

Linguagem: Python (Ambiente integrado com SQL via SQLAlchemy).

Análise de Dados: Pandas para tratamento de tabelas e cruzamento de informações.

Visualização: Seaborn e Matplotlib para geração de relatórios gráficos.

📂 Estrutura do Projeto
1. Estruturação da Tabela
O projeto inicia com a organização das bases de dados da Meteora (Itens, Pedidos, Produtos e Vendedores), garantindo que as informações relevantes estejam prontas para consulta.

2. Análise de Desempenho do Time
Com o objetivo de bonificar a equipe em 2021, filtramos e comparamos as vendas de 2019 e 2020.

Identificação de Talentos: Quem são os vendedores com maior volume e consistência?

Critério de Bônus: Cruzamento entre quantidade de pedidos e valor total vendido.

3. Foco Regional: São Paulo (SP)
A Meteora acredita no potencial do mercado paulista. Nossa análise respondeu a duas perguntas críticas:

São Paulo já rende muitas vendas em relação aos outros estados?

Quais são os dois vendedores que mais dominam o mercado da região?

📈 Resultados e Insights
Crescimento Identificado: As análises confirmaram que 2020 apresentou uma evolução positiva em relação a 2019, permitindo uma margem maior para bônus salariais.

Liderança de SP: Através de consultas SQL, validamos a concentração de mercado em São Paulo, justificando o incremento de investimentos no estado.

Seleção de Especialistas: Identificamos os profissionais com maior aderência ao público paulista para liderar a estratégia de expansão focada em 2021.
